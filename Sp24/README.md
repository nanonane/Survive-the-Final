## 数据库系统

### 考试题型

- 基本概念（40分）：选择、填空、判断、概念
- 应用题（60分），以下每一条都会占8-15分：
  - 基本关系代数（选择、投影、连接、笛卡尔积等）；基本SQL编程（基本查询、聚集查询、嵌套查询）
  - 触发器
  - 给出需求，画E-R图（基本，含2-3个实体及1-3个联系），E-R图转化为关系模式
  - 函数依赖（公理及推论证明），模式分解（无损连接、保持依赖），范式与规范化（3NF, BCNF）
  - 查询优化
  - 事务处理
